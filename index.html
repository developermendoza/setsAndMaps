Sets and Maps

Sets:

A Set is a collection of unique values where each value must be distinct. In other words, it doesn't allow duplicate elements. Sets are iterable, and the order of elements is based on insertion order

Maps:

A Map is a collection of key-value pairs where each key must be unique. Unlike objects, keys in a Map can be of any data type, including objects and primitive values. Maps are iterable, and the order of elements is based on insertion order.
